## 声明响应式状态

[官方介绍](https://cn.vuejs.org/guide/essentials/reactivity-fundamentals.html)：可以使用`reactive()`函数创建一个响应式对象或数组

```js
import {reactive} from 'vue'
const state = reactive({count:0})
```
